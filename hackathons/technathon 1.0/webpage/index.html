ABOUT

The idea of the project is finding misplaced objects such as Spectacles, pens, house/car Keys, laptops and everything which can hold a small strip of plastic and vinyl on it.

We will be using sensors which shall be placed as a tag on the objects to be detected.

So whenever someone misplaces their objects and are unable to find them this technology will be quite helpful. We shall be placing tiny sensors on some tags which will be wrapped on the objects and hence can be detected easily as and when required.

All that a person has to do is just buy these tags , wrap them around objects which are more likely to be misplaced (like keys, spectacles, pens, torchlights,..etc.)

PROJECT

This Tag is a component which allows an Arduino board to connect to the internet and operate using an mobile application. It also allows the objects to be sensed and control remotely across existing network. This can be done using IoT and motion sensors.
